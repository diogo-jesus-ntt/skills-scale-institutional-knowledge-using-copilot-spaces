# OctoAcme Project Management Docs

This folder contains OctoAcme's project management process documentation for planning and delivering work consistently across teams. The process follows a practical lifecycle: initiate the project with clear goals and stakeholders, plan scope and milestones, execute and track delivery with quality checks, manage risks and communications, release safely, and use retrospectives to drive continuous improvement.
